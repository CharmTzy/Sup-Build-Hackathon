## Change Reports

Every time an agent changes this repository, add a short report in `CHANGES/`.
Use the filename format `YYYY-MM-DD_HH-MM-SS_short-task-name.md` with local time.
Each report must include what changed, what is finished, what is still missing,
and what can be improved for the hackathon demo.
